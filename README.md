# UE4 Voxels

<b>Requires 4.17 (async collision cooking)</b>

Based on [transvoxel](http://transvoxel.org/)

## Features:
* Real time collisions
* Smooth LOD
* Fast (update of a chunk cost less than 1ms)
* Blueprint interface
* Textures
* Multithreaded
* Easy generation of custom worlds
* Free and open source
* Multiplayer
* Landscape import
* Easy load/save
* Up to 256 materials
* Water
* Dynamic slicing
